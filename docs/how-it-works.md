# Nilakantha級数による円周率(π)の計算の仕組み

このプログラムは、**Nilakantha（ニラカンタ）級数** と呼ばれる無限級数を使って、円周率（π）の近似値を計算しています。

## Nilakantha級数とは？

Nilakantha級数は、以下のような形で表される、πに収束する級数です。

π = 3 + 4/(2*3*4) - 4/(4*5*6) + 4/(6*7*8) - 4/(8*9*10) + ...

この級数は、最初の「3」に、符号が交互に変わる分数を次々と足し引きしていくことで、πに近づいていきます。

## なぜこのプログラムは動くのか？

プログラムの `calculatePi` 関数は、このNilakantha級数をコンピュータで計算できるようにしたものです。

1.  **`let estimate = 3;`**
    *   計算の出発点として、級数の最初の項である `3` を設定します。

2.  **`while (Math.abs(estimate - Math.PI) > targetError)`**
    *   このループは、計算の「終わり」を決めるための条件です。
    *   `estimate`（現在のπの推定値）と、JavaScriptが知っているπの真の値 (`Math.PI`) との差の絶対値が、あらかじめ設定した `targetError`（許容誤差）よりも大きい間、計算を繰り返します。
    *   つまり、「求める精度に達するまで計算を続ける」という処理を行っています。

3.  **`const term = 4 / (i * (i + 1) * (i + 2));`**
    *   これがNilakantha級数の分数の部分です。
    *   `i` は `2, 4, 6, 8, ...` と2ずつ増えていき、級数の各項の分母 `(2*3*4)`, `(4*5*6)`, `(6*7*8)` を作り出しています。

4.  **`estimate += sign * term;`**
    *   計算した分数 (`term`) を、`estimate` に加算または減算します。
    *   `sign` は `1` と `-1` が交互に切り替わるようになっており、これにより級数の `+` と `-` を表現しています。

## まとめ

このプログラムは、Nilakantha級数の数式を忠実に再現し、それをコンピュータで繰り返し計算することで、円周率（π）を目標とする精度で求めることができるのです。
