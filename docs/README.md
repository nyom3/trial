# 円周率(π) 計算プログラム

これは、Nilakantha級数を用いて円周率（π）の近似値を計算するTypeScriptプログラムです。

## 概要

このプロジェクトは、指定された精度に基づいて円周率の値を推定します。
計算ロジックは `pi.ts` に記述されており、そのアルゴリズムの解説は `docs/how-it-works.md` を参照してください。

## 使い方

1.  **依存関係のインストール:**

    ```bash
    npm install
    ```

2.  **ビルド:**

    ```bash
    npm run build
    ```

3.  **実行:**

    ```bash
    node dist/pi.js
    ```

    実行すると、コンソールに推定された円周率の値が出力されます。

## テスト

このプロジェクトには、計算ロジックを検証するためのテストが含まれています。

```bash
npm test
```
